CSS Flexbox 完全指南

Flexbox（弹性盒子布局）是 CSS3 中引入的一种强大的布局方式，它让容器能够灵活地调整其子元素的大小、顺序和对齐方式，非常适合构建响应式布局。

什么是 Flexbox？

Flexbox 是一种一维布局方法，用于在行或列中排列元素。它提供了更有效的方式来分配空间、对齐和对齐容器中的项目。

基本概念

Flexbox 布局由两个主要组件组成：

  • Flex 容器（Flex Container）：设置了 display: flex 或 display: inline-flex 的元素
  • Flex 项目（Flex Items）：Flex 容器的直接子元素

容器属性

flex-direction

定义主轴方向，决定项目的排列方向：

  • row：默认值，从左到右
  • row-reverse：从右到左
  • column：从上到下
  • column-reverse：从下到上

justify-content

定义项目在主轴上的对齐方式：

  • flex-start：默认值，左对齐
  • flex-end：右对齐
  • center：居中
  • space-between：两端对齐，项目之间间隔相等
  • space-around：每个项目两侧间隔相等
  • space-evenly：项目之间和两端间隔都相等

align-items

定义项目在交叉轴上的对齐方式：

  • stretch：默认值，拉伸填满容器
  • flex-start：交叉轴起点对齐
  • flex-end：交叉轴终点对齐
  • center：交叉轴中点对齐
  • baseline：基线对齐

flex-wrap

定义项目是否换行：

  • nowrap：默认值，不换行
  • wrap：换行，第一行在上方
  • wrap-reverse：换行，第一行在下方

项目属性

flex-grow

定义项目的放大比例，默认为 0（不放大）。如果所有项目的 flex-grow 都为 1，则它们将等分剩余空间。

flex-shrink

定义项目的缩小比例，默认为 1（可缩小）。如果空间不足，项目会缩小。

flex-basis

定义在分配多余空间之前，项目占据的主轴空间。默认值为 auto，即项目的本来大小。

align-self

允许单个项目有与其他项目不一样的对齐方式，可覆盖 align-items 属性。

实际应用示例

Flexbox 非常适合用于：

  • 导航栏布局
  • 卡片网格布局
  • 居中内容
  • 响应式设计
  • 表单布局

总结

Flexbox 是现代 CSS 布局的核心工具之一。掌握这些属性后，您将能够轻松创建各种复杂的布局。虽然现在有了 Grid 布局，但 Flexbox 仍然是一维布局的最佳选择。
